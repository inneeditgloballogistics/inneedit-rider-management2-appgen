   ├─ Notification sent to Hub Manager (new_rider_onboarding)

✅ Hub Manager sees new rider in dashboard
   ├─ Views "New Riders" tab
   ├─ Searches for rider
   └─ Clicks "Start Handover"

✅ Hub Manager completes handover form
   ├─ Enters odometer reading
   ├─ Selects fuel level
   ├─ Uploads vehicle photos (1-3)

   └─ Clicks "Complete Handover"

✅ System creates handover record
   ├─ Stores all details and photos
   ├─ Saves rider signature
   ├─ Creates handover_vehicle record
   └─ Sends notification to rider

✅ Rider receives congratulation notification
   ├─ Sees "Vehicle Handed Over Successfully!"

   └─ Heads to assigned store
